Fruit Frequency

Everything in this game is made to be simple and elegant to change: If you want to change the frequency of the fruits as well as the levels you can simply open the settings.swift from the left sidebar of Xcode. Inside at the top you will see this: 

You can adjust from here depending on which level you are how many fruits to throw. Of course the user will not be able to see the level number but the fruits will increase depending his score. It is recommented not to exceed 10 fruits ! The second part is to adjust the levels. If the user scores more then 4 then it will move to the next level and so on. It is that simple to adjust the fruits. Later on we will see how to change the graphics.

 

Blade Adjustment

You can adjust the thickness of the blade. To do that again we need to go to Settings.swift

 

At the top you can see that you can change two values. The first one is the scale of the blade as you go lower the blade will go smaller and the opposite. You MUST know that if you are going to make the blade smaller you MUST change the blade radius as well. 

Graphics Replacement

Finally at this point where you can reskin the game. There are two options of reskinning. The preffered one is to open the folder that containts all the graphics and just replace the images with yours and of course with the same name and extension. Other method is to drag and drop the new images into the project and open the settings.swift at the very bottom you will see the strings of the images: 

From here you can adjust the font name and the graphics for the fruits and the bomb. Each fruit consist of image of the full fruit and two images for the sliced fruit left/right part. Replace the strings with the one that you justed drag and drop and run the game. It is that simple yes :) 

 

Music Replacement

The best way to change the music is to open the folder that have all the sounds: 

and start replacing the sounds with yours. Make sure they have the same name and extension. IT IS REQUIRED TO CHANGE THE MUSIC SINCE THESE ARE THE ORIGINALS SOUNDS OF FRUIT NINJA AND YOU DON'T WANT TO HAVE DEAL WITH HALF BRICKS

Icons and Splash Screens

Select the project from Xcode:

 

Go to general tab:

Scroll Down until you see this: 

Click on the small arrow next to AppIcon and you must see this: 

 

You must change all  the icons from here. You can drag and drop icons from your desctop directly to the icon boxes. REMEMBER !!!!! that the sizes is in pt. for example the first icon is 29pt BUT there are 3 cions there. This is because you need to multiply the 29pt and the preffix 1x 2x 3x to get the pixels. For example the first icon is 29pt * 1x = 29px the second one is 29pt * 2x = 58px and so on for the others. 

 

The same goes for the splash screens
